## 📋 Sobre o projeto

Portfólio profissional desenvolvido como atividade avaliativa da disciplina de **Programação Web** do curso de **Tecnólogo em Sistemas para Internet** da **UESPI**.

---

## ✅ Requisitos atendidos

- ✔️ Estrutura semântica com `<header>`, `<section>` e `<footer>`
- ✔️ Menu de navegação com links para todas as seções
- ✔️ Seção **Sobre Mim** com informações pessoais e habilidades
- ✔️ Seção **Meus Projetos** renderizada dinamicamente via JavaScript
- ✔️ Array de objetos em JavaScript com os projetos
- ✔️ Estilização com **Flexbox** e **Grid**
- ✔️ Responsividade para dispositivos móveis (menu hamburguer)
- ✔️ Publicado via **GitHub Pages**
